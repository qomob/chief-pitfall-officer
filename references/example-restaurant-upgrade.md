# 完整示例：餐饮门店品牌形象升级

本示例展示从需求澄清到终稿输出的完整流程，作为输出质量和格式的参考基准。

---

## 第一阶段：需求澄清

### 1. 初始模糊需求
> **用户输入**："我们是一家开了3年的中式快餐店，想升级一下门店形象，提升档次，吸引更多年轻客群。预算有限，大概下个月就要动工。"

### 2. 顾问针对性提问
1. **资金与时间约束**：您的"预算有限"具体金额是多少（如：10万还是30万）？下个月动工，期望的停业改造天数上限是多少？
2. **升级范围**：本次升级是否包含品牌VI（Logo、餐具包装、菜单）的重设，还是仅针对SI（门店空间、软装、灯光）的重构？
3. **门店现状**：现有门店面积多大？厨房排烟、给排水等重度硬件是否需要变动，还是只做前厅的"轻装修"？
4. **客群定位**：您提到的"年轻客群"，更偏向于上班族白领的快速就餐，还是年轻人的社交打卡？

### 3. 用户反馈整合
> **用户回复**：预算在15万元以内；停业时间不能超过7天；需要重新设计Logo和餐具菜单，前厅空间和门头重做，厨房格局不变；客群定位是18-35岁的年轻上班族，希望风格是"国潮简约风"。

### 4. 正式工作目标声明

*   **项目名称**：经典中式快餐"国潮简约风"品牌与空间一体化升级项目
*   **核心目标**：在 15 万元预算内，通过 VI 重塑（门头、包装、菜单）与前厅 SI 轻度改造，提升品牌年轻化感知；在 7 天停业期内完成施工，确保下月顺利开业。
*   **关键约束**：预算≤15万，停业≤7天，不动后厨格局
*   **成功标准**：门头视觉焕新率100%，前厅空间打卡率提升，开业首周客流环比+20%

## 方案选型

| 方案 | 核心思路 | 优势 | 劣势 | 适用条件 |
| :--- | :--- | :--- | :--- | :--- |
| **方案A（推荐）：轻度改造+工厂预制** | VI重塑+前厅轻装（免漆板/集成墙板）+门头工厂预制现场拼装 | 成本低（≤15万），施工快（7天），不停业期间可同步设计 | 视觉效果受"轻装"限制，无法做结构改动 | 预算≤15万，停业≤7天，不动后厨 |
| **方案B：中度改造+现场施工** | VI重塑+前厅中度硬装（含墙面重做、局部电路改造） | 视觉效果更好，可做更多定制化设计 | 成本可能超预算（18-20万），施工需10-12天 | 预算>15万，停业可接受10天以上 |

**推荐方案**：方案A。推荐理由：用户预算≤15万且停业≤7天是硬约束，方案B在这两个维度都存在超限风险。

## 七维系统拆解

### 一、工作内容拆解（WBS）

```
[项目总任务：门店形象升级]
├── 阶段一：品牌视觉（VI）重塑（第1-10天）
│    ├── 子任务1.1：国潮简约风Logo及标准色设计
│    ├── 子任务1.2：菜单、餐具、外卖包装延展设计
│    └── 子任务1.3：灯箱片及店内海报文案设计
├── 阶段二：空间（SI）轻量化设计（第5-15天）
│    ├── 子任务2.1：门头及前厅动线布局规划（保留厨房）
│    ├── 子任务2.2：灯光及软装点缀方案（主打出片、提亮）
│    └── 子任务2.3：施工节点图与材料清单（BOM）输出
├── 阶段三：现场施工与软装导入（第20-27天，停业7天）
│    ├── 子任务3.1：旧门头及前厅饰面拆除（1天）
│    ├── 子任务3.2：新门头安装与前厅基础硬装（3天）
│    ├── 子任务3.3：灯具、软装及桌椅摆放（2天）
│    └── 子任务3.4：保洁与空气治理（1天）
└── 阶段四：新装开业与营销（第28-30天）
     ├── 子任务4.1：员工新VI围裙/工服配发与服务培训
     └── 子任务4.2：线上美团/大众点评视觉同步更新及开业促销
```

### 二、所需技能梳理

| 技能类别 | 必备技能项 | 具体应用场景 |
| :--- | :--- | :--- |
| **技术/专业技能** | 2D品牌视觉设计（Illustrator/PS） | Logo、包装、菜单及海报物料设计 |
| | 3D空间设计（CAD/3ds Max/Sketchup） | 前厅动线、灯光布局及门头效果图绘制 |
| | 商业照明设计 | 利用暖色调灯光（3000K-3500K）提升食物食欲感 |
| **通用/软技能** | 供应链与材料采购 | 寻找高性价比的国潮风软装、桌椅及灯具供应商 |
| | 工程项目管理 | 严格控制 7 天施工工期，协调各工种交叉作业 |
| **行业专项技能** | 餐饮动线设计（客流与服务员动线） | 确保收银、取餐、返还餐具动线不交叉，提升坪效 |
| | 餐饮消防与环保合规 | 确保新门头与前厅改建符合消防安全及城管招牌规范 |

### 三、岗位职能匹配

*   **项目经理（店长/创始人兼任）**
    *   *职能*：整体进度把控、预算审批、各方协调。
    *   *协作*：主导全局，审批设计与施工方案。

*   **品牌平面设计师（外包/第三方）**
    *   *职能*：负责 Logo、菜单、包装等所有视觉传达物料。
    *   *协作*：配合空间设计师，确保空间内的视觉元素与 VI 一致。

*   **空间设计师（第三方/施工队自带）**
    *   *职能*：负责门头效果图、前厅布局、灯光及材料选型。
    *   *协作*：向施工队长进行技术交底。

*   **施工队长及工人（专业餐饮工装团队）**
    *   *职能*：负责拆除、硬装施工、强弱电改造及软装安装。
    *   *协作*：在项目经理监督下，严格按图纸在 7 天内完工。

### 四、流程动线梳理

```
【前期准备：15天】
[VI设计完成] ──(依赖)──> [SI空间设计完成] ──> [施工图会审与材料预购]
                                               │
                                               ▼
                                       ★ [关键决策点：材料到货确认]
                                               │
                                               ▼
【施工闭店：7天】                          [交底与闭店准备]
[Day 1: 拆除与垃圾清运] ──> [Day 2-3: 吊顶/墙面基层与电路改造] ──> [Day 4-5: 门头安装/涂料刷漆]
                                                                         │
                                                                         ▼
[Day 7: 开业准备/保洁] <── [Day 6: 灯具安装/桌椅软装进场] <──────────────┘
         │
         ▼
【正式开业】
```

**关键决策点说明**：第 15 天的"施工图与材料会审"。必须确保所有定制材料（如发光字门头、特定尺寸桌椅）在闭店前全部到货，否则决不能盲目闭店拆除。

### 五、交付成果界定

| 阶段 | 交付成果物 | 格式/形态 | 质量/验收标准 |
| :--- | :--- | :--- | :--- |
| **视觉设计** | 品牌视觉规范手册（VI Manual） | PDF 电子档 | 包含标准色（RGB/CMYK）、主Logo、辅助图形，放大缩小不失真 |
| **空间设计** | 施工图纸及效果图 | CAD（.dwg）/ PDF | 包含平面布局图、天花灯具图、门头立面图，尺寸标注偏差 < 5mm |
| **包装物料** | 实物菜单与餐具包装样片 | 纸质实物 / 打样盒 | 菜单字迹清晰、无色差；餐具包装材质符合食品级安全标准 |
| **空间交付** | 完工门店前厅与门头 | 实体空间 | 门头灯箱正常发光，前厅无明显异味，灯光无频闪，动线无阻碍 |

### 六、成本预算规划（总预算：15万元）

#### 1. 预算分配比例
*   **设计费用（VI+SI）**：1.5 万元（10%）
*   **门头及前厅硬装施工**：8 万元（53%）
*   **灯具与软装桌椅采购**：3.5 万元（23%）
*   **包装/菜单首批物料印刷**：1 万元（7%）
*   **机动预备金**：1 万元（7%）

#### 2. 费用管控规则
*   **限额设计**：要求设计师在方案设计阶段，必须使用市面常见规格的材料（如标准尺寸防火板、生态板），避免定制异形件以压缩材料成本。
*   **保留非必要变动**：后厨格局及设备100%保留不动；地面瓷砖若无破损，仅做深度清洁，不进行重铺。

### 七、可行性评估

*   **时间可行性（风险高，需重点防控）**：
    - *评估*：7 天闭店施工期非常紧张，任何湿作业（如水泥地面找平、大面积乳胶漆自然阴干）都会导致延期。
    - *对策*：墙面采用快装集成墙板或免漆板，避免现场刷漆；门头钢结构及发光字在闭店前于工厂预制完毕，现场仅进行拼装。

*   **资金可行性（可行）**：
    - *评估*：15 万元对于 100 平米以内的快餐店进行"轻装升级"是充裕的，但前提是不动结构、不动后厨。
    - *对策*：严格执行"重装饰、轻装修"策略，靠国潮风挂画、霓虹灯置景、特色餐具来营造氛围。

*   **可行性结论**：**方案基本可行**。项目成功的核心在于**"工厂预制+现场拼装"**，必须做好闭店前的物料筹备工作，方可确保 7 天内无缝交付。
