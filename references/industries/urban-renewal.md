# 城市更新/旧改行业适配指南

### 典型业务场景
- 老旧小区外立面及管网改造
- 工业遗存转文化创意产业园
- 历史文化街区活化利用
- 城市微空间/口袋公园建设

### 方案选型决策树
```
[城市更新判断]
├── 综合整治类
│   ├── 居住区 → 基础基建+软装提升方案
│   └── 商业区 → 业态重塑+动线优化方案
└── 功能改变类
    ├── 工业转办公 → 结构加固+空间切割方案
    └── 历史转商业 → 保护性修缮+沉浸式体验方案
```

### 环境核查要点
- **技术**：老旧建筑的基础承重、水电容量及管网现状。
- **政策**：历史建筑保护红线、规划局审批流程。
- **供应链**：老旧材料（如古法青砖、特定型号木材）的供应稳定性。

### 参与方风险预警
- **角色**：测绘/勘察方。**风险**：原始图纸与现状不符，导致设计方案无法落地。
- **角色**：施工方。**风险**：在旧建筑中施工引发不可预见的结构损坏。
- **法务/财务预警**：合同需包含“不可预见地质/结构条件”的责任分担协议；明确因“居民投诉”导致停工的工期顺延及损失补偿条款。
- **防甩锅方案**：开工前进行全量三维激光扫描建模；约定“不可预见费”的使用边界。

### 验收关键项
- **材料**：加固材料的检测报告（碳纤维、灌注胶）。
- **执行时间**：严格避开居民休息时段的施工时间记录。
- **交付评估**：改造后公共空间的使用率及居民满意度测评。
