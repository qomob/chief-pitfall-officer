# 甲方首席防坑官 (Chief Pitfall Officer, CPO)

> **"让甲方在专业套路面前，不再是‘待宰的羔羊’。"**

`chief-pitfall-officer` 是一款专门为**甲方（项目发起方）**量身打造的 AI 智能审计与需求拆解技能。它深度集成了法务、财务、技术、管理四大领域的实战经验，旨在帮助那些在特定行业缺乏经验的决策者识别乙方套路、规避合作风险、填补知识盲区。

---

## 🛡️ 核心价值

在项目落地前，CPO 充当您的“影子智囊团”，提供以下保障：
- **破解知识盲区**：自动识别行业隐藏的环境门槛（政策、技术、供应链）。
- **穿透成本黑盒**：提供行业基准报价，审计人工、材料、损耗的合理性。
- **阻断合同陷阱**：深度审计 IP 归属、违约赔偿、分包限制等法务条款。
- **管控执行偏差**：设定关键决策点 (★) 签字制度，确保过程不走样。

---

## 🏗️ 十维全链路保护体系

不同于普通的任务拆解，CPO 强制执行十个维度的深度审计：

1.  **WBS 任务拆解**：消除执行过程中的“黑盒”环节。
2.  **材料审计**：防范以次充好，锁定主材品牌与规格。
3.  **人工审计**：核查资历真实性，防止人力虚报。
4.  **参与方风险**：预警各角色推诿场景，划分权责边界。
5.  **环境核查**：识别政策红线、技术约束及市场门槛。
6.  **量化验收**：将模糊形容词转化为可测量的硬指标。
7.  **报价审核**：揭露隐性收费，提供成本审计 Checklist。
8.  **过程管控**：设定阶段性签字节点与现场审计点。
9.  **法务合规**：审计合同条款，保护知识产权与退出权利。
10. **财务结算**：优化付款节奏 (2-3-3-2)，锁定维保成本。

---

## 👥 谁需要 CPO？

- **中小企业主 (SME)**：缺乏专业采购团队，需要全能型避坑指南。
- **企业采购/审计官**：需要跨行业项目的专业审计基准。
- **跨界项目负责人**：临时负责不熟悉领域（如行政负责数字化转型）的经理。
- **政企项目协调人**：面临极高合规压力，需确保专款专用。

---

## 📈 已适配的热门行业

技能库内置了以下高风险行业的深度适配方案：
- **餐饮/门店改造**：环评、消防、城管审批及动线审计。
- **直播带货**：GMV 对赌、主播风险、佣金结算基数审计。
- **城市更新/旧改**：历史建筑保护、不可预见费、结构安全审计。
- **医疗大健康**：GSP 认证、数据隐私、耗材“打印机模式”审计。
- **金融科技/政务数字化**：信创要求、渗透测试、资金专户管理审计。
- **项目策划/展览展示**：咨询顾问资历、多媒体兼容性、现场封样审计。
