## 2. 核心成本审计维度

### A. 人工审计 (Labor Audit)
- **资历溢价核实**：要求乙方提供核心人员的行业成功案例及近 3 个月社保证明，防止“挂名专家、新手执行”。
- **人天配比审查**：审计项目经理 (PM) 与执行人员 (Staff) 的比例。标准比例应在 1:3 到 1:5 之间；比例过高（如 1:10）意味着管理失控，过低则可能存在人力虚报。
- **隐性人工成本**：警惕“夜间施工加急费”、“节假日补助”是否已包含在基础单价中。

### B. 材料审计 (Materials Audit)
- **品牌替换风险**：主材必须明确品牌、系列、规格（如“多乐士金装五合一”而非“大品牌乳胶漆”）。
- **损耗率核算**：行业标准损耗通常在 5%-8% 之间。如果乙方在报价单中列出 15% 以上的损耗，属于变相加价。
- **非标件询价**：针对“定制艺术装置”、“异形柜体”，要求乙方提供至少两家上游加工厂的原始报价单或合同存根。

### C. 时间成本审计 (Time Audit)
- **关键路径分析 (CPM)**：要求乙方提供甘特图。审查“养护期”（如水泥找平）、“审批期”等刚性时间是否预留充足。
- **并行工期挤压**：警惕乙方为了中标承诺不可能完成的工期，最终通过降低材料规格或减少工艺步骤来赶工。

## 3. 过程管理与交付评估

### A. 过程管理审计点
- **关键决策节点 (★)**：每个阶段结束必须有甲方签字的《阶段性验收单》，未签字不得进行下一阶段。
- **例会制度**：要求周报中必须包含“本周费用执行情况”与“下周预算预警”。
- **现场审计**：针对工程类项目，引入第三方监理进行不定期的隐蔽工程（如水电管网）抽检。

### B. 交付评估模型 (3D Model)
- **Dimension 1: 功能/物理指标**：是否达到合同约定的技术规格？
- **Dimension 2: 业务/效果指标**：如直播间的转化率、门店的客流量提升、策划案的品牌覆盖率。
- **Dimension 3: 移交/资产指标**：所有源代码、设计原稿、供应商名录、操作手册是否已完整移交？

## 4. 乙方常见收费陷阱 (Redlines)

| 陷阱名目 | 表现形式 | 破解之道 |
| :--- | :--- | :--- |
| **人头虚报** | 报价单上写着 10 人开发，实际只有 3 人干活。 | 要求乙方提供核心成员社保证明或项目专属钉钉群人数核对。 |
| **重复收费** | 软件许可费已含基础模块，又单独列出“接口费”。 | 要求拆解报价细项，对比功能点与收费项的映射关系。 |
| **隐性收费** | 合同签完后才告知：服务器、短信费、三方插件需另付。 | 合同约定“交钥匙”工程，明确列出所有不包含的第三方项。 |
| **维护费陷阱** | 首年免费，次年维护费高达合同额的 30%。 | 行业标准为 10%-15%，需在主合同中锁死后续维保单价。 |
| **管理费虚高** | 在物料费基础上加收 20% 以上的“综合管理费”。 | 明确管理费涵盖的具体服务，并与行业平均水平（10%-15%）对比。 |

## 3. 报价审核 Checklist

- [ ] 报价单是否包含：单价、数量、单位、总价、备注说明？
- [ ] 软硬件报价是否与市场公开价对比过（如京东、阿里云官网）？
- [ ] 人力成本报价是否与当地平均薪资水平匹配？
- [ ] 是否存在“包干项”与“实报实销项”的混淆？
- [ ] 税率是否明确（专票/普票）？
- [ ] 是否已剔除不合理的“加班费”、“不可预见费”？

---

## 4. 成本核算小工具 (逻辑)

**合理总成本 = (人力天数 × 日薪) + 硬件/第三方采购 + 合理利润 (15%-25%) + 风险预备金 (5%-10%)**

*如果乙方报价偏离此逻辑超过 30%，需重点审核其工作量评估的真实性。*
