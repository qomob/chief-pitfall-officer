# 医疗大健康行业适配指南

### 典型业务场景
- 智慧医院/数字化病房建设
- 医疗器械与精密仪器采购
- 医药冷链仓储与物流体系
- 互联网医院/在线问诊平台

### 方案选型决策树
```
[医疗需求判断]
├── 数字化建设
│   ├── 基础型 → HIS/LIS 系统升级+标准硬件
│   └── 智慧型 → AI 辅助诊断+物联网病房全案
└── 供应链/冷链
    ├── 自建模式 → GSP 认证冷库+自有车队方案
    └── 外包模式 → 三方冷链监管+全链路数据追溯
```

### 环境核查要点
- **政策**：医疗器械经营许可证（二类/三类）、GSP 认证时效。
- **技术**：医疗数据安全性（等保三级）、PACS 影像存储冗余度。
- **环境**：实验室/冷库的温湿度恒定报警系统及 UPS 备用电源。

### 参与方风险预警
- **角色**：软件厂商。**风险**：数据接口封闭，与现有医保/卫健系统无法对接。
- **角色**：维保商。**风险**：精密设备停机响应慢，导致医疗事故风险。
- **法务/财务预警**：严格审计“数据安全协议”，明确患者隐私泄露的无限赔偿责任；财务上锁定耗材的阶梯采购价，防止设备低价、耗材天价的“打印机模式”。
- **防甩锅方案**：合同约定“开放接口协议”及“故障赔付条款”；设立 24 小时现场响应机制。

### 验收关键项
- **材料**：进口设备的报关单及原产地证明。
- **合规**：临床验证报告或软件安全评估证书。
- **交付评估**：系统上线后 30 天无宕机运行记录。

### 落地执行蓝图 (医疗大健康)

#### 1. 前期需求调研清单 (甲方自查)
- [ ] **资质盘点**：确认本机构持有的医疗器械经营许可证类别（一类/二类/三类），及 GSP 认证有效期。
- [ ] **数据合规**：确认现有系统是否通过等保三级测评，患者数据存储是否满足《个人信息保护法》要求。
- [ ] **接口现状**：梳理与医保、卫健委、电子健康档案等外部系统的接口协议及版本号。
- [ ] **场地条件**：确认机房/实验室的温湿度控制、UPS 备用电源、消防设施是否达标。

#### 2. 服务商评标标准
- **资质权重 (25%)**：医疗器械经营许可、ISO 13485（医疗器械质量管理体系）、等保三级测评资质。
- **经验权重 (35%)**：近 2 年有 3 个以上同类型（如智慧医院/互联网医院）落地案例，需提供可验证的验收报告。
- **接口开放度 (20%)**：是否承诺开放 HL7/FHIR 标准接口，拒绝私有协议锁定。
- **售后响应 (20%)**：是否承诺 24 小时现场响应，精密设备是否有备机替换机制。

#### 3. 执行节点与巡检计划
- **需求确认阶段 (★核心)**：确认乙方提供的功能清单与甲方临床流程完全匹配，重点核对急诊/手术等关键科室的特殊需求。
- **系统集成阶段 (★核心)**：三方联调（乙方系统 + 医保接口 + 原有 HIS/LIS），必须在测试环境完成全流程闭环验证。
- **试运行阶段**：选择 1-2 个科室先行上线，运行 2-4 周，记录故障率及临床反馈。
- **正式上线阶段**：全科室切换，乙方团队驻场 1-2 周保障。

#### 4. 合规资质办理指引
- **医疗器械注册/备案**：涉及二类器械需省级药监局备案，三类需国家药监局注册。
- **等保测评**：系统上线前必须通过等保三级测评，测评周期约 2-3 个月。
- **GSP 认证**：涉及药品/器械仓储的，需通过 GSP 认证，含温湿度监控系统验收。
- **互联网医院牌照**：如涉及线上问诊，需向省级卫健委申请互联网医院执业许可。

#### 5. 应急预案 (Plan B)
- **风险 1：系统上线后频繁宕机** → 触发阈值：月度宕机 > 2 次或单次 > 4 小时。补救：启动备用系统，乙方免费驻场整改，超期按合同额 0.5%/天赔付。
- **风险 2：接口无法对接医保/卫健系统** → 触发阈值：联调 3 次仍失败。补救：要求乙方自费聘请接口适配方，或甲方有权更换供应商。
- **风险 3：精密设备故障导致临床停摆** → 触发阈值：核心设备停机 > 8 小时。补救：合同约定备机 24 小时内到位，停机期间乙方承担替代方案费用。

#### 6. 验收与上线筹备清单
- [ ] **系统验收**：30 天无重大故障运行记录，核心业务链路响应时间 < 3 秒。
- [ ] **数据验收**：患者数据 100% 完整迁移，脱敏规则校验通过。
- [ ] **培训验收**：各科室操作培训完成，培训签到表归档。
- [ ] **文档移交**：系统架构文档、接口文档、运维手册、应急预案手册完整移交。
