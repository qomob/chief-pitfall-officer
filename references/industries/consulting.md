# 项目策划/创意全案适配指南

### 典型业务场景
- 城市品牌形象全案策划
- 文旅项目整体定位与招商
- 大型公关活动/节庆策划
- 企业数字化转型战略咨询

### 方案选型决策树
```
[策划需求判断]
├── 品牌定位类
│   ├── 快速版 → 调研+命名+视觉系统方案
│   └── 深度版 → 顶层设计+战略地图+落地督导方案
└── 营销推广类
    ├── 节点式 → 单次活动+媒体矩阵方案
    └── 长期式 → 年度代运营+内容飞轮方案
```

### 环境核查要点
- **市场**：目标受众画像的颗粒度是否支持执行？
- **政策**：策划方案是否符合地方产业政策导向？

### 参与方风险预警
- **角色**：咨询顾问。**风险**：交付的是“通用模板”，缺乏行业实操性。
- **角色**：执行外包。**风险**：创意落地打折扣，物料制作粗糙。
- **防甩锅方案**：阶段性交付评审；约定咨询人员的现场驻点天数。

### 验收关键项
- **人工**：项目经理的行业案例背景核实。
- **过程管理**：周报制度及关键创意节点的比稿记录。
- **交付评估**：方案对核心业务目标的贡献度（ROI 预估）。
