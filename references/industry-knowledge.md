# 环境核查与行业知识库

本文档为甲方提供不同行业的环境核查清单，识别技术、政策、供应链、市场四大环境中的隐藏风险点。

---

## 1. 环境分类标准

| 环境维度 | 关注重点 | 甲方易忽略点 |
| :--- | :--- | :--- |
| **技术环境** | 基础设施、兼容性、扩展能力 | 存量数据清洗成本、接口私有协议限制、服务器扩容阶梯价 |
| **政策环境** | 行业准入、数据合规、地方补贴 | 临时性环保限产、跨境数据出境审批、行业资质年审窗口期 |
| **供应链环境** | 物料交期、供应商依赖、物流成本 | 关键零部件唯一供应商风险、原材料价格波动传导、非标件打样周期 |
| **市场环境** | 竞争动态、用户习惯、渠道门槛 | 竞品突发价格战、流量平台算法黑盒、线下渠道进场费及账期 |

## 2. 行业专属环境核查清单 (Checklist)

### 互联网/数字化转型
- [ ] **技术**：是否涉及旧系统迁移？（旧数据格式不规范会导致 30%+ 额外工作量）
- [ ] **技术**：三方接口（支付、地图、短信）是否已开通且实名？
- [ ] **政策**：App/小程序是否已完成备案？是否涉及金融/医疗敏感数据？
- [ ] **市场**：应用商店上架是否有特殊资质要求（如 ICP 证、文网文）？

### 制造业/产线升级
- [ ] **技术**：厂房配电功率是否支撑新设备同时运行？
- [ ] **政策**：设备进口是否涉及关税减免或反倾销调查？
- [ ] **供应链**：设备维保备件是否可在国内 24 小时内获取？
- [ ] **环境**：地基承重及抗震等级是否符合精密仪器安装要求？

### 餐饮/新零售
- [ ] **政策**：门头招牌高度、亮度是否符合当地城管最新视觉要求？
- [ ] **政策**：后厨排污、排烟系统是否通过环保部门分级认证？
- [ ] **供应链**：核心食材是否有 2 家以上稳定供货商以规避季节性断货？
- [ ] **市场**：商圈内同质化竞争对手的日均翻台率及外卖占比基线？

### 医疗/大健康
- [ ] **政策**：涉及二类/三类医疗器械经营，相关仓储环境是否符合 GSP 规范？
- [ ] **技术**：系统是否符合等级保护（等保三级）要求？
- [ ] **供应链**：试剂/耗材的冷链物流是否有完整温控数据追溯？

### 服务业/教育培训
- [ ] **政策**：预收费资金管理是否已接入政府指定监管平台？
- [ ] **技术**：直播/点播带宽峰值在万人同时在线时是否有冗余？
- [ ] **市场**：行业广告禁用词清单是否已同步至营销团队？

## 3. 跨行业知识标签系统

当需求输入后，自动匹配以下标签以调用对应知识：
- `#DataCompliance`: 涉及用户隐私、跨境数据。
- `#HardwareConstraint`: 涉及场地、电力、承重、安规。
- `#SupplyChainDependency`: 涉及长交期物料、单一供应商。
- `#LicensingRedline`: 涉及办学许可、办医许可、特种设备许可。
