# 房地产/建筑行业适配指南

### 典型业务场景
- 住宅/商业项目开发全流程
- 精装房标准化体系搭建
- 营销中心/样板房建设
- 旧改/城市更新项目
- 物业管理体系搭建

### 方案选型决策树
```
[需求类型判断]
├── 新项目开发
│   ├── 住宅 → 标准：拿地→规划→设计→施工→预售→交付
│   ├── 商业 → 需增加：招商→运营→管理环节
│   └── 旧改 → 需增加：拆迁安置→居民沟通→改造方案
├── 营销中心/样板房
│   ├── 预算<100万 → 标准化装修+软装方案
│   └── 预算>100万 → 定制设计+沉浸式体验方案
└── 物业体系
    ├── 新建物业 → 制度→人员→系统→服务标准全流程
    └── 物业升级 → 差距分析+专项整改+系统导入
```

### 拆解侧重点
设计规划、工程管理、合规审批、资金管理

### 常见约束
规划审批周期（30-90天）、天气影响施工、资金周转、政策变动

### 关键角色
建筑师、结构工程师、总包单位、监理、销售经理

### 预算特征
- 新项目：土建 40-50%，精装 20-30%，设计费 5-8%，营销 3-5%
- 营销中心：装修 40-50%，家具软装 20%，AV设备 10%
- 物业体系：人力 50-60%，系统 15-20%，培训 10%

### 行业红线
- 规划许可证/施工许可证（无证不得开工）
- 竣工验收备案（未验收不得交付使用）
- 环评/能评（开工前必须完成）
- 农民工工资专户（强制要求）
- 预售资金监管（售房款必须进入监管账户）
- 消防设计审查与验收
