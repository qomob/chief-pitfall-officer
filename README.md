# 甲方首席防坑官 (Chief Pitfall Officer, CPO)

> **"风险规避 + 落地执行 + 自进化：让甲方在专业套路面前不再是'待宰的羔羊'。"**

`chief-pitfall-officer` 是一款专门为**甲方（项目发起方）**量身打造的 AI 智能审计与全流程落地技能。它深度集成了法务、财务、技术、工程管理四大领域的实战经验，旨在帮助那些在特定行业缺乏经验的决策者识别乙方套路、规避合作风险，并提供从需求梳理到落地执行的可操作 SOP 指南。

**与静态 Prompt 不同，CPO 内置自进化闭环——每次使用都会沉淀知识，用得越多越精准。**

---

## 核心价值：三维完整体系

### 1. 风险规避 (Pitfall Avoidance)
- **破解知识盲区**：自动识别行业隐藏的环境门槛（政策、技术、供应链）。
- **穿透成本黑盒**：提供行业基准报价，审计人工、材料、损耗的合理性。
- **阻断合同陷阱**：深度审计 IP 归属、违约赔偿、分包限制等法务条款。

### 2. 落地执行 (Execution Blueprint)
- **全周期 SOP**：提供前期调研、服务商招标、合规办理、竣工验收的全套执行清单。
- **关键巡检计划**：设定施工/执行过程中的关键巡检点（★），确保过程不走样。
- **业务就绪清单**：覆盖从物理交付到业务上线（开业/发布）的最后 100 米动作。

### 3. 自进化闭环 (Self-Evolution Loop)
- **运行时记录**：每次方案输出后自动记录执行数据（行业、场景、风险命中数）。
- **知识增量提炼**：从用户反馈中提炼新套路、新法规、新价格等知识增量。
- **知识库自动更新**：验证后的知识直接写入行业知识库，下次执行自动生效。
- **vNext 种子传递**：知识更新后生成 Creator 指令种子，显式注入下次执行——这是从"学习"到"自进化"的关键跨越。
- **版本追踪**：version.json 作为 SSOT 记录每次变更，支持回归对比。
- **审计追踪**：所有知识库修改均有日志记录，可追溯每一次进化。

## 十维全链路审计 + 六阶段执行蓝图

### 十大审计维度
1. **WBS 任务拆解** | 2. **材料与人工审计** | 3. **参与方风险图谱** | 4. **流程动线核查** | 5. **量化验收手册**
6. **报价审计与成本** | 7. **过程管控点 (★)** | 8. **交付评估模型** | 9. **法务合规审计** | 10. **财务结算审计**

### 六大执行阶段（多 Agent 协作）
```
[录入] → 复杂度分级 → vNext种子注入 → Fan-out → Risk Auditor(阶段一～二) → Execution Planner(阶段三) → Quality Validator(阶段四) → Synthesizer(阶段五) → Evolution Recorder(阶段六)
```
- **阶段一**（Risk Auditor）：行业识别、需求澄清、环境核查
- **阶段二**（Risk Auditor）：方案对比选型、十维甲方保护拆解
- **阶段三**（Execution Planner）：六模块执行蓝图
- **阶段四**（Quality Validator）：五维风险评估 + WBS 逻辑自检 + 质量审计(L3) + 回归对比(L3.5) + 优化建议(L4)
- **阶段五**（Synthesizer 你）：终稿整合输出
- **阶段六**（Evolution Recorder）：运行时记录 → 知识增量 → 知识库更新 → vNext 种子生成

## 谁需要 CPO？

- **中小企业主 (SME)**：缺乏专业采购团队，需要全能型避坑与执行指南。
- **企业采购/审计官**：需要跨行业项目的专业审计基准与执行标准。
- **跨界项目负责人**：临时负责不熟悉领域（如行政负责数字化转型）的经理。
- **政企项目协调人**：面临极高合规压力，需确保项目按时按质按规落地。

## 快速上手

### 触发词
您可以直接输入以下内容触发：
- `"帮我理理需求"`
- `"怎么防坑"`
- `"怎么落地这个项目"`
- `"乙方报价合理吗"`
- `"甲方首席防坑官，帮我看看这个全案"`

### 输入示例
> "连锁咖啡品牌 小红书种草，预算 10 万，如何防止被广告公司坑？"
> "我想开一家 200 ㎡的火锅店，预算 50 万，请出具一份完整的防坑及落地执行方案。"

### 自进化反馈示例
使用方案后，您可以说：
> "上次方案里没考虑到消防通道宽度要求"
> "实际装修价格是 1800 元/㎡，和你说的 1200-1500 不一样"
> "乙方用了新套路：先低价中标再通过增项加价"

这些反馈会被自动提炼为知识增量，更新到知识库中。
