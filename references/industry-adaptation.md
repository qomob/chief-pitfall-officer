# 行业适配指南

不同行业的拆解侧重点差异很大。本指南包含每个行业的：典型业务场景、方案选型决策树、拆解侧重点、行业红线。

---

## 餐饮/零售行业

### 典型业务场景
- 门店品牌形象升级（VI+SI改造）
- 新店开业全流程策划
- 菜单/产品线调整与定价策略
- 外卖业务搭建与运营体系
- 连锁门店标准化体系搭建

### 方案选型决策树

```
[需求类型判断]
├── 形象升级类
│   ├── 预算<10万 → 轻装改造方案（软装+灯光+VI延展）
│   ├── 预算10-30万 → 中度改造方案（门头+前厅+VI重塑）
│   └── 预算>30万 → 全面翻新方案（VI+SI+设备+营销全链路）
├── 新店开业类
│   ├── 单店 → 标准开店SOP（选址→设计→施工→招训→开业）
│   └── 连锁首店 → 建立标准化模板+首店试点+复制模型
└── 运营优化类
    ├── 线上为主 → 外卖平台运营体系搭建
    └── 线下为主 → 堂食动线优化+翻台率提升方案
```

### 拆解侧重点
空间设计、供应链、合规审批、客户体验

### 环境核查要点
- **政策**：门头招牌高度、亮度是否符合当地城管最新视觉要求？
- **政策**：后厨排污、排烟系统是否通过环保部门分级认证？
- **供应链**：核心食材是否有 2 家以上稳定供货商以规避季节性断货？
- **市场**：商圈内同质化竞争对手的日均翻台率及外卖占比基线？

### 参与方风险预警
- **角色**：施工队长。**风险**：低价中标后通过隐性增项收费。
- **角色**：设计师。**风险**：设计图纸与现场实际尺寸不符，导致返工。
- **防甩锅方案**：所有设计改动必须经由监理方签字，并保留现场测量原始数据。

### 验收关键项
- **功能**：收银系统压力测试（单店 100 笔/分钟不掉线）。
- **性能**：排烟系统最大功率下噪音 ≤ 65dB。
- **合规**：食品级装修材料认证文件。

### 常见约束
停业天数、城管审批、消防合规、食品安全标准

### 关键角色
空间设计师、施工队长、品牌设计师、供应链经理

### 预算特征
- 轻装修：软装/灯光 40%，VI设计 20%，营销 15%
- 重装修：硬装施工 50%+，软装/设备 20-30%
- 新店开业：租金押金 30%，装修 25%，设备 20%，首批物料 10%

### 行业红线
- 食品安全相关材料和设备必须持证（食品级认证）
- 门头招牌需城管审批（周期 7-15 天）
- 消防通道不能改动
- 后厨排烟系统改动需环保审批
- 餐饮废弃物处理需有合规渠道

## 电商/互联网行业

### 典型业务场景
- 独立站搭建（Shopify/自建站）
- 私域流量体系搭建（企业微信+小程序）
- 跨境电商多平台运营体系
- 会员体系设计与落地
- 数据埋点与分析体系搭建

### 方案选型决策树

```
[需求类型判断]
├── 建站类
│   ├── 无技术团队 → SaaS方案（Shopify/有赞/微盟）
│   ├── 有技术团队 → 开源方案（WooCommerce/Magento）或自研
│   └── 跨境为主 → Shopify + 海外仓方案
├── 流量/获客类
│   ├── 预算<5万/月 → 内容营销+SEO长线方案
│   ├── 预算5-20万/月 → 投放+内容组合方案
│   └── 预算>20万/月 → 全渠道投放+KOL矩阵方案
└── 数据/分析类
    ├── 埋点 → GA4/Mixpanel/神策（按数据量级选型）
    └── BI → Metabase/Superset（轻量）或 Tableau（重量）
```

### 拆解侧重点
产品迭代、用户增长、数据埋点、技术架构

### 环境核查要点
- **技术**：是否涉及旧系统迁移？（旧数据格式不规范会导致 30%+ 额外工作量）
- **技术**：三方接口（支付、地图、短信）是否已开通且实名？
- **政策**：App/小程序是否已完成备案？是否涉及金融/医疗敏感数据？
- **市场**：应用商店上架是否有特殊资质要求（如 ICP 证、文网文）？

### 参与方风险预警
- **角色**：乙方开发团队。**风险**：核心架构师在中途离职，新人接手导致代码质量下降。
- **角色**：第三方云服务商。**风险**：流量峰值时服务器带宽被限速。
- **防甩锅方案**：合同约定代码必须每日提交至甲方 Git 库，并定期进行第三方代码审计。

### 验收关键项
- **功能**：核心支付链路 100% 成功率测试。
- **性能**：GA4 埋点数据延迟 < 10s。
- **合规**：个人信息保护法 (PIPL) 合规性自查报告。

### 常见约束
开发周期、技术栈限制、用户量级、数据合规

### 关键角色
产品经理、技术负责人、运营负责人、数据分析师

### 预算特征
- SaaS建站：平台费 20%，主题/插件 15%，内容/设计 30%，推广 25%
- 自研系统：人力成本 60%+，服务器/工具 15-20%，测试 10%
- 运营推广：投放费 50-70%，内容制作 15-20%，工具/数据 10%

### 行业红线
- 数据隐私合规（个人信息保护法、GDPR 如涉及海外）
- 支付接口需持牌（或使用合规第三方支付）
- 上线需灰度发布（不能全量直接上线）
- 核心链路必须有回滚方案
- 用户数据不得未经授权共享给第三方

## 制造业

### 典型业务场景
- 自动化产线引进与改造
- 质量管理体系升级（ISO/IATF）
- MES/ERP系统导入
- 新产品试产到量产（NPI）
- 车间精益改善（5S/TPM）

### 方案选型决策树

```
[需求类型判断]
├── 产线改造类
│   ├── 预算<50万 → 局部自动化（单工位机器人/工装改善）
│   ├── 预算50-200万 → 半自动产线（关键工序自动化+人工辅助）
│   └── 预算>200万 → 全自动产线（机器人+视觉+MES集成）
├── 系统导入类
│   ├── 中小厂（<200人） → 轻量MES/ERP（如简道云/钉钉宜搭）
│   ├── 中型厂（200-1000人） → 标准ERP（如用友/金蝶）
│   └── 大型厂（>1000人） → 重量ERP（如SAP/Oracle）
└── 质量升级类
    ├── 现有体系完善 → 差距分析+专项整改
    └── 新建体系 → 全流程导入（文件→培训→试运行→外审）
```

### 拆解侧重点
产线改造、质量控制、供应链管理、安规认证

### 环境核查要点
- **技术**：厂房配电功率是否支撑新设备同时运行？
- **政策**：设备进口是否涉及关税减免或反倾销调查？
- **供应链**：设备维保备件是否可在国内 24 小时内获取？
- **环境**：地基承重及抗震等级是否符合精密仪器安装要求？

### 参与方风险预警
- **角色**：设备供应商。**风险**：非标件研发失败导致交付延期。
- **角色**：安装团队。**风险**：施工破坏现有产线，导致大面积停产。
- **防甩锅方案**：约定“阶梯式验收”，每完成一个工位即签收一个工位；购买“第三方责任险”。

### 验收关键项
- **功能**：设备空载运行 72 小时无故障。
- **性能**：产线良率 ≥ 99.8%。
- **合规**：特种设备检验合格证。

### 常见约束
停产窗口期、设备交期、安规认证周期、现有体系不能断档

### 关键角色
工艺工程师、设备供应商、质量主管、生产主管

### 预算特征
- 产线改造：设备采购 40-60%，安装调试 15-20%，培训 5%
- 系统导入：软件许可 30-40%，实施服务 30%，培训 10%
- 质量升级：咨询费 30%，培训 20%，检测设备 25%

### 行业红线
- 安全生产许可证（涉及特种设备的必须持证）
- 环评审批（新建/改建产线必须）
- 设备验收需第三方检测（如特种设备检验所）
- ISO/IATF 体系不能断档（外审周期内必须维持）
- 危化品使用需单独审批和存储

## 教育/培训行业

### 典型业务场景
- 新课程体系研发与上线
- 线上教学平台搭建
- 校区/培训中心筹建
- 招生营销体系搭建
- 师资培养与认证体系

### 方案选型决策树

```
[需求类型判断]
├── 课程研发类
│   ├── 线下为主 → 课程大纲→教案→试讲→迭代→正式开课
│   ├── 线上为主 → 课程大纲→录播/直播→题库→上线运营
│   └── 混合式 → 线上内容+线下翻转课堂方案
├── 平台搭建类
│   ├── 预算<10万 → SaaS平台（小鹅通/Classin）
│   └── 预算>10万 → 定制开发或企业版SaaS
└── 校区筹建类
    ├── 租赁改造 → 选址→装修→设备→招师→招生
    └── 自建 → 规划→审批→建设→验收→运营
```

### 拆解侧重点
课程研发、师资配置、招生渠道、教学体验

### 环境核查要点
- **政策**：预收费资金管理是否已接入政府指定监管平台？
- **技术**：直播/点播带宽峰值在万人同时在线时是否有冗余？
- **市场**：行业广告禁用词清单是否已同步至营销团队？

### 参与方风险预警
- **角色**：核心讲师。**风险**：项目中途跳槽并带走核心教案。
- **角色**：推广服务商。**风险**：使用违规营销手段导致品牌受损或账号被封。
- **防甩锅方案**：签署详尽的《知识产权归属协议》；约定营销素材需经甲方审核后方可发布。

### 验收关键项
- **功能**：学员完课率数据自动统计功能。
- **性能**：直播课时延 < 500ms。
- **合规**：办学许可证在有效期内，备案信息一致。

### 常见约束
学期时间节点、场地容量、师资资质、教材审核周期

### 关键角色
课程总监、招生负责人、教务主管、技术支持

### 预算特征
- 课程研发：师资 40%+，内容制作 25%，测试验证 10%
- 平台搭建：平台费 30%，内容迁移 20%，运营培训 15%
- 校区筹建：装修 35%，设备 25%，师资储备 15%，营销 15%

### 行业红线
- 办学许可证（K12/职业教育/高等教育的许可要求不同）
- 教师资格证（授课教师必须持证）
- 课程内容需审核备案（特别是涉及意识形态的内容）
- 收费需纳入监管账户（预收费管理）
- 广告宣传不得使用"保过""包就业"等绝对化用语

## 活动策划/会展行业

### 典型业务场景
- 企业年会/客户答谢会
- 产品发布会
- 行业峰会/论坛
- 展览展台设计与搭建
- 线上直播活动

### 方案选型决策树

```
[需求类型判断]
├── 企业年会/答谢会
│   ├── 预算<20万 → 酒店宴会厅+标准AV+自助餐方案
│   ├── 预算20-50万 → 定制创意+沉浸式体验方案
│   └── 预算>50万 → 场地定制搭建+全案创意+明星/KOL方案
├── 产品发布会
│   ├── 线下为主 → 场地+AV+媒体邀约+直播方案
│   └── 线上为主 → 直播平台+虚拟演播厅+社交媒体矩阵
└── 展览展台
    ├── 标摊（9-18㎡） → 标准搭建+基础物料方案
    └── 特装（>36㎡） → 定制设计+互动装置+接待方案
```

### 拆解侧重点
创意方案、供应商管理、现场执行、客户体验

### 环境核查要点
- **政策**：大型活动（200人+）需向辖区公安分局报批。
- **政策**：场地消防验收必须通过。
- **技术**：现场 AV 系统是否有主备链路？
- **环境**：搭建材料需达到防火等级认证（B1级以上）。

### 参与方风险预警
- **角色**：搭建商。**风险**：使用劣质材料导致结构坍塌或异味。
- **角色**：AV 供应商。**风险**：关键时刻音响/大屏黑屏且无备机。
- **防甩锅方案**：要求提供材料防火检测报告；现场彩排必须包含“断电/断网”应急演练。

### 验收关键项
- **功能**：现场直播流推流成功。
- **性能**：舞台承重测试（按最大演职人员数 1.5 倍冗余）。
- **合规**：公安安保审批回执。

### 常见约束
活动日期固定不可改、场地档期、搭建撤展时间、安全容量

### 关键角色
创意总监、项目统筹、AV技术、搭建商、客户经理

### 预算特征
- 年会/答谢会：场地餐饮 25-35%，AV 15-20%，搭建 15-20%，创意 8-10%
- 发布会：场地 20%，AV/直播 25%，媒体 15%，创意 15%
- 展览展台：设计搭建 40-50%，AV 15%，运营人员 15%，物流 10%

### 行业红线
- 大型活动（200人+）需向辖区公安分局报批
- 场地消防验收必须通过
- 现场必须有应急预案（含医疗、疏散、设备故障）
- 搭建材料需达到防火等级认证（B1级以上）
- 涉及外籍人员需办理邀请函和签证手续

## 房地产/建筑行业

### 典型业务场景
- 住宅/商业项目开发全流程
- 精装房标准化体系搭建
- 营销中心/样板房建设
- 旧改/城市更新项目
- 物业管理体系搭建

### 方案选型决策树

```
[需求类型判断]
├── 新项目开发
│   ├── 住宅 → 标准：拿地→规划→设计→施工→预售→交付
│   ├── 商业 → 需增加：招商→运营→管理环节
│   └── 旧改 → 需增加：拆迁安置→居民沟通→改造方案
├── 营销中心/样板房
│   ├── 预算<100万 → 标准化装修+软装方案
│   └── 预算>100万 → 定制设计+沉浸式体验方案
└── 物业体系
    ├── 新建物业 → 制度→人员→系统→服务标准全流程
    └── 物业升级 → 差距分析+专项整改+系统导入
```

### 拆解侧重点
设计规划、工程管理、合规审批、资金管理

### 常见约束
规划审批周期（30-90天）、天气影响施工、资金周转、政策变动

### 关键角色
建筑师、结构工程师、总包单位、监理、销售经理

### 预算特征
- 新项目：土建 40-50%，精装 20-30%，设计费 5-8%，营销 3-5%
- 营销中心：装修 40-50%，家具软装 20%，AV设备 10%
- 物业体系：人力 50-60%，系统 15-20%，培训 10%

### 行业红线
- 规划许可证/施工许可证（无证不得开工）
- 竣工验收备案（未验收不得交付使用）
- 环评/能评（开工前必须完成）
- 农民工工资专户（强制要求）
- 预售资金监管（售房款必须进入监管账户）
- 消防设计审查与验收

## 直播带货行业

### 典型业务场景
- 品牌自播间从 0 到 1 搭建
- 达人直播全案策划
- 618/双11 大促直播运营
- 直播间动线设计与设备配置
- 短视频+直播联动引流方案

### 方案选型决策树
```
[直播需求判断]
├── 品牌自播
│   ├── 预算<10万 → 手机直播+基础灯光+兼职主播方案
│   ├── 预算10-50万 → 虚拟演播室/实景基地+专业AV+全职团队方案
│   └── 预算>50万 → 品牌旗舰店+多机位+明星/头部达人联动
└── 达人分销
    ├── 坑位费模式 → 纯佣/低坑位+高佣方案
    └── 专场模式 → 深度内容定制+全渠道预热
```

### 环境核查要点
- **技术**：直播间上行带宽是否独享（建议 ≥ 20Mbps）？
- **政策**：是否涉及敏感类目（医疗、金融）需要报白？
- **市场**：竞品直播间的开播时段及核心话术策略。

### 参与方风险预警
- **角色**：代运营公司 (DP)。**风险**：承诺 GMV 无法达成，通过“注水”数据骗取佣金。
- **角色**：主播。**风险**：临场罢工或违反职业道德导致账号被封。
- **法务/财务预警**：严审“佣金结算基数”，必须扣除退货、优惠券及运费后的实付额；约定“肖像权”及“直播切片”的授权期限。
- **防甩锅方案**：合同约定 GMV 对赌协议，佣金与扣除退货后的结算额挂钩。

### 验收关键项
- **人工**：核心场控、运营、主播的简历审核及现场试讲。
- **时间**：脚本排练时间、开播前 1 小时设备联调。
- **材料**：直播间装修材料的吸音效果测试（背景墙、隔音棉）。
