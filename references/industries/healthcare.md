# 医疗大健康行业适配指南

### 典型业务场景
- 智慧医院/数字化病房建设
- 医疗器械与精密仪器采购
- 医药冷链仓储与物流体系
- 互联网医院/在线问诊平台

### 方案选型决策树
```
[医疗需求判断]
├── 数字化建设
│   ├── 基础型 → HIS/LIS 系统升级+标准硬件
│   └── 智慧型 → AI 辅助诊断+物联网病房全案
└── 供应链/冷链
    ├── 自建模式 → GSP 认证冷库+自有车队方案
    └── 外包模式 → 三方冷链监管+全链路数据追溯
```

### 环境核查要点
- **政策**：医疗器械经营许可证（二类/三类）、GSP 认证时效。
- **技术**：医疗数据安全性（等保三级）、PACS 影像存储冗余度。
- **环境**：实验室/冷库的温湿度恒定报警系统及 UPS 备用电源。

### 参与方风险预警
- **角色**：软件厂商。**风险**：数据接口封闭，与现有医保/卫健系统无法对接。
- **角色**：维保商。**风险**：精密设备停机响应慢，导致医疗事故风险。
- **法务/财务预警**：严格审计“数据安全协议”，明确患者隐私泄露的无限赔偿责任；财务上锁定耗材的阶梯采购价，防止设备低价、耗材天价的“打印机模式”。
- **防甩锅方案**：合同约定“开放接口协议”及“故障赔付条款”；设立 24 小时现场响应机制。

### 验收关键项
- **材料**：进口设备的报关单及原产地证明。
- **合规**：临床验证报告或软件安全评估证书。
- **交付评估**：系统上线后 30 天无宕机运行记录。
