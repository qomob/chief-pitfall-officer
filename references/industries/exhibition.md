# 展览展示/门店改造适配指南

### 典型业务场景
- 企业展厅设计与施工
- 品牌连锁店 SI 标准化改造
- 博物馆/规划馆展项策划
- 临时快闪店设计与搭建

### 方案选型决策树
```
[展陈/改造判断]
├── 长期展厅
│   ├── 预算<5000/㎡ → 静态图文+标准展柜方案
│   └── 预算>8000/㎡ → 多媒体交互+定制艺术装置方案
└── 连锁店改造
    ├── 翻新模式 → 视觉延展+局部软装方案
    └── 拆改模式 → 空间重构+机电系统升级方案
```

### 参与方风险预警
- **角色**：多媒体集成商。**风险**：软件与硬件不兼容，现场调试延误开馆。
- **角色**：软装代采商。**风险**：实物与效果图色差过大，材质以次充好。
- **防甩锅方案**：要求核心设备先行进场联调；所有主材必须经甲方现场封样。

### 验收关键项
- **价格费用**：材料单价对比市场公开价，严审“定制件”报价。
- **材料**：木工板、环保胶水的甲醛释放量检测报告。
- **交付评估**：门店改造后的动线流畅度测试（顾客停留时长分析）。
