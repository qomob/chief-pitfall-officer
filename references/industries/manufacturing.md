# 制造业适配指南

### 典型业务场景
- 自动化产线引进与改造
- 质量管理体系升级（ISO/IATF）
- MES/ERP系统导入
- 新产品试产到量产（NPI）
- 车间精益改善（5S/TPM）

### 方案选型决策树
```
[需求类型判断]
├── 产线改造类
│   ├── 预算<50万 → 局部自动化（单工位机器人/工装改善）
│   ├── 预算50-200万 → 半自动产线（关键工序自动化+人工辅助）
│   └── 预算>200万 → 全自动产线（机器人+视觉+MES集成）
├── 系统导入类
│   ├── 中小厂（<200人） → 轻量MES/ERP（如简道云/钉钉宜搭）
│   ├── 中型厂（200-1000人） → 标准ERP（如用友/金蝶）
│   └── 大型厂（>1000人） → 重量ERP（如SAP/Oracle）
└── 质量升级类
    ├── 现有体系完善 → 差距分析+专项整改
    └── 新建体系 → 全流程导入（文件→培训→试运行→外审）
```

### 拆解侧重点
产线改造、质量控制、供应链管理、安规认证

### 环境核查要点
- **技术**：厂房配电功率是否支撑新设备同时运行？
- **政策**：设备进口是否涉及关税减免或反倾销调查？
- **供应链**：设备维保备件是否可在国内 24 小时内获取？
- **环境**：地基承重及抗震等级是否符合精密仪器安装要求？

### 参与方风险预警
- **角色**：设备供应商。**风险**：非标件研发失败导致交付延期。
- **角色**：安装团队。**风险**：施工破坏现有产线，导致大面积停产。
- **防甩锅方案**：约定“阶梯式验收”，每完成一个工位即签收一个工位；购买“第三方责任险”。

### 验收关键项
- **功能**：设备空载运行 72 小时无故障。
- **性能**：产线良率 ≥ 99.8%。
- **合规**：特种设备检验合格证。
