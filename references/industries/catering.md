# 餐饮/零售行业适配指南

### 典型业务场景
- 门店品牌形象升级（VI+SI改造）
- 新店开业全流程策划
- 菜单/产品线调整与定价策略
- 外卖业务搭建与运营体系
- 连锁门店标准化体系搭建

### 方案选型决策树
```
[需求类型判断]
├── 形象升级类
│   ├── 预算<10万 → 轻装改造方案（软装+灯光+VI延展）
│   ├── 预算10-30万 → 中度改造方案（门头+前厅+VI重塑）
│   └── 预算>30万 → 全面翻新方案（VI+SI+设备+营销全链路）
├── 新店开业类
│   ├── 单店 → 标准开店SOP（选址→设计→施工→招训→开业）
│   └── 连锁首店 → 建立标准化模板+首店试点+复制模型
└── 运营优化类
    ├── 线上为主 → 外卖平台运营体系搭建
    └── 线下为主 → 堂食动线优化+翻台率提升方案
```

### 拆解侧重点
空间设计、供应链、合规审批、客户体验

### 环境核查要点
- **政策**：门头招牌高度、亮度是否符合当地城管最新视觉要求？
- **政策**：后厨排污、排烟系统是否通过环保部门分级认证？
- **供应链**：核心食材是否有 2 家以上稳定供货商以规避季节性断货？
- **市场**：商圈内同质化竞争对手的日均翻台率及外卖占比基线？

### 参与方风险预警
- **角色**：施工队长。**风险**：低价中标后通过隐性增项收费。
- **角色**：设计师。**风险**：设计图纸与现场实际尺寸不符，导致返工。
- **防甩锅方案**：所有设计改动必须经由监理方签字，并保留现场测量原始数据。

### 验收关键项
- **功能**：收银系统压力测试（单店 100 笔/分钟不掉线）。
- **性能**：排烟系统最大功率下噪音 ≤ 65dB。
- **合规**：食品级装修材料认证文件。

### 落地执行蓝图 (餐饮门店装修版)

#### 1. 前期需求调研清单 (甲方自查)
- [ ] **物业底单**：获取原始结构图、给排水图、强弱电系统图。
- [ ] **容量核查**：确认配电房额定功率、排烟管道管径是否满足新设备需求。
- [ ] **营商环境**：确认商场/街道对装修材料运输、建筑垃圾清运的特殊规定。

#### 2. 服务商评标标准
- **资质权重 (20%)**：建筑装饰二级、消防工程二级。
- **经验权重 (40%)**：近 2 年有 3 个以上同商圈或同类型餐饮落地案例。
- **工期承诺 (20%)**：是否具备预制件施工能力，停业天数是否满足 7-15 天极限挑战。
- **售后保障 (20%)**：在本地是否有 2 小时响应的紧急维修团队。

#### 3. 施工节点与巡检计划
- **进场阶段**：现场放线复核，核对隐蔽工程点位与设计图是否冲突。
- **水电阶段 (★核心巡检)**：水管 0.8Mpa 压力测试（30min 不降压），电线品牌规格抽检（杜绝非标线）。
- **木工阶段**：吊顶龙骨密度检查，免漆板边角处理工艺巡检。
- **安装阶段**：门头发光字固定牢靠度，收银台强弱电接口位置复核。

#### 4. 合规资质办理指引
- **消防申报**：300㎡以上必须办理消防设计备案及竣工验收。
- **环评登记**：涉及重油烟需安装符合标准的油烟净化器并保留第三方清洗记录。
- **招牌审批**：施工前需向城管委提交效果图备案，审批通过方可搭建围挡。

#### 5. 开业筹备清单 (最后 100 米)
- **工程验收**：甲醛深度治理、电路满负荷运行测试（模拟全开设备 4 小时）。
- **业务筹备**：美团/点评视觉换新、收银系统压力测试、员工服务动线演练。
- **证照就绪**：营业执照、食品经营许可证现场公示。
